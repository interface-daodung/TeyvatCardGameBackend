# Admin Web Frontend

React + TypeScript + Vite admin interface for the Teyvat Card Game.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:8000`

## Build

```bash
npm run build
```

## Features

- Dashboard with statistics and charts
- User management (view, ban/unban, manage currency)
- Payment tracking
- Game data management (characters, equipment, adventure cards, maps)
- Localization management
- Audit logs
